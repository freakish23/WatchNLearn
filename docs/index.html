How to test

  1. Download and unzip the demo build from Releases (or use the Download button above).
  2. Open Chrome → chrome://extensions/ → enable Developer mode → Load unpacked → choose the unzipped folder.
  3. Open one of the demo videos and wait for the overlay to appear at the timestamps listed below.

Notes for testers

  • Use full YouTube links (not shortened `youtu.be`) so the content script matches correctly.
  • If the overlay doesn't appear, open DevTools and look for `[YT-Overlay]` logs in the page console and `[Overlay UI]` logs in the iframe console.
  • The demo stores responses locally in Chrome storage, nothing is sent to any server.
